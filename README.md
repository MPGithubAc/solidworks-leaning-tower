# solidworks-leaning-tower
Originally made for a midterm in visual dynamics, this version of the leaning tower of pisa showcases my work in solidworks and my ability to navigate the tools to create simulations.

The tower is functionally composed of 3 distinct pieces: a separate "bottom" section of the tower which showcases the section that is closest to the ground, a "middle" section which is duplicated to create the center, and a "top" piece which is placed at the crown of the edifice. Included in the project files is a simulation of the tower inside of a wind tunnel, which helps showcase the structural stability of my work.
